CSS селектори. Визначення і види

CSS селектори визначають до якого елемента застосовувати те чи інше CSS правило. Зверніть увагу, не існує селекторів, якіб дозволили вибрати батька (контейнер що містить) або сусіда батька, або нащадків сусіда батька.

Базові селектори

Універсальний селектор - обирає всі елементи. За бажанням, він може бути обмежений певним простором імен, або стосуватися всього простору імен

  • Синтаксис: * ns|* *|*

  • Приклад: * буде відповідати всім елементам на сторінці.

Селектори за типом елемента

Цей базовий селектор обирає тип елементів, до яких буде застосовуватися правило.

  • Синтаксис: елемент

  • Приклад: селектор input выберет все элементы input.

Селектори за класом

Цей базовий селектор обирає елементи,грунтуючись на значенні їх атрибута class

  • Синтаксис: .ім'яКласу

  • Приклад: селектор .index обере всі елементи з відповідним класом (який був визначений в атрибуті class="index").

Селектори за ідентифікатором

Цей базовий селектор оибирає елементи, ґрунтуючись на значенні їхнього id атрибута. Не забувайте, що ідентифікатор повинен бути унікальним, тобто використовуватися тільки для одного елемента в HTML-документі.

  • Синтаксис: #ім'яІдентифікатора

  • Приклад: селектор #toc обере елемент із ідентифікатором toc (який був визначений в атрибуті id="toc").

Селектори за атрибутом

Це селектор обирає всі елементи, що мають цей атрибут або атрибут з певним значенням

  • Синтаксис: [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value]

  • Приклад: селектор [autoplay] обере всі елементи, які мають атрибут autoplay (незалежно від його значенні)..
